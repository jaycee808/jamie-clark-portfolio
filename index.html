02
projects.

PEEKTIONARY

Peektionary is an image guessing game powered by openAI's DallE image generator and ChatGPT. This interactive game generates a random image based on a prompt word and users have to guess the word or words used to generate the image.

Built using React and styled with Tailwind CSS, Peektionary utilises the Random Slug Generator NPM module to generate prompts for the DallE API.

ARTREK

Artrek is an application that combines the Europeana API and Google Maps API to provide users with an easy search experience. Simply enter a specific art term and the app returns a comprehensive set of results. With Google Map locations, users can easily navigate to the museum or collection where the artefact is stored.

The app was built using HTML, CSS, and JavaScript.

WEATHER DASHBOARD

This application is powered by the OpenWeatherMap API, it retrieves detailed weather data for today and forecast for the next five days.

Built with HTML, CSS, Bootstrap, & JavaScript.

PASSWORD GENERATOR

This app allows users to generate a random password based on their preference of length and characters to use. Once the user has selected their preferred characters, the application generates a secure password.

Built with HTML, CSS, Bootstrap, & JavaScript.

SIDEKICK

Sidekick is an app to keep track of daily to-dos. It incorporates Moment.js to dynamically generate the date.

Built with HTML, CSS, JavaScript, jQuery, & Moment.js.
